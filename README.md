# AI Study Buddy

## About the Project

AI Study Buddy is an intelligent study assistant designed to help students learn more effectively. It can explain complex topics in simple terms, summarize detailed notes into structured points, generate multiple-choice quizzes for self-assessment, and create flashcards for quick revision.

The goal of the project is to make studying faster, clearer, and more interactive using AI-powered content generation.

## Technologies Used

- Python
- Streamlit
- Groq API
